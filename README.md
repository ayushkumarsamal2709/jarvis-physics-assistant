# Jarvis Physics Assistant 🎤⚛️

## 📌 Overview

Jarvis Physics Assistant is a **voice-controlled desktop assistant** built using Python. It can execute system commands and answer **physics-related questions** using voice interaction.

---

## 🎯 Features

* 🎤 Voice command recognition
* 🔊 Text-to-speech response
* 🌐 Open websites (Google, etc.)
* 💻 Launch applications
* ⚛️ Physics Q&A system

---

## 🧠 Physics Mode

The assistant can answer questions like:

* "Explain Schrödinger equation"
* "Define entropy"
* "What is wave function?"
* "Explain uncertainty principle"

---

## ⚙️ Tech Stack

* Python
* SpeechRecognition
* pyttsx3

---

## 🚀 How to Run

1. Install dependencies:

```
pip install -r requirements.txt
```

2. Run:

```
python main.py
```
